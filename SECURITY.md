# 安全政策

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 最新版 | ✅ 完全支持 |
| < 1.0 | ⚠️ 最佳 effort |

## 报告安全漏洞

如果你发现了安全漏洞，请通过以下方式私下报告：

📧 邮箱：security@synapse-ai.dev

请不要在公开的 Issue 中披露漏洞细节。

我们会在 48 小时内回复，并在 7 天内提供修复计划。

## 安全最佳实践

1. **API 密钥管理**：不要将 API 密钥提交到代码仓库
2. **本地运行**：默认只在 localhost 运行 Gateway
3. **Skill 审核**：仔细检查第三方 Skill 的代码
4. **定期更新**：保持依赖项更新到最新版本
