### 【场景 4】交互式聊天（60-105 秒）

**画面：**
- 分屏：左边终端对话，右边实时显示 Token 消耗
- 输入问题，AI 流式回复

```
synapse> 帮我写一个 Python 脚本，批量重命名文件

AI: 好的，这是一个使用 os 模块的脚本：

import os

def batch_rename(folder, old_ext, new_ext):
    for filename in os.listdir(folder):
        if filename.endswith(old_ext):
            new_name = filename.replace(old_ext, new_ext)
            os.rename(
                os.path.join(folder, filename),
                os.path.join(folder, new_name)
            )
            print(f"Renamed: {filename} -> {new_name}")

📝 Token 消耗: 156 input / 89 output
```

**解说词：**
> "来试试基础功能。让 AI 写个批量重命名脚本，可以看到，这次对话只用了不到 300 Token。同样的请求在 Claude Code 里可能要上千 Token。"

- 💬 自然语言交互

## 🎨 制作指南

### 视觉风格

| 元素 | 规格 |
|------|------|
| 主色调 | `#3B82F6` (蓝色) + `#10B981` (绿色) |
| 背景 | 深色主题 `#0F172A` |
| 字体 | JetBrains Mono (代码), Inter (正文) |
| 动画 | 流畅的淡入淡出，避免花哨效果 |

### 推荐工具

| 用途 | 工具 |
|------|------|
| 屏幕录制 | OBS Studio / Screen Studio |
| 终端录制 | asciinema / terminalizer |
| 视频剪辑 | DaVinci Resolve / Final Cut Pro |
| 动效设计 | After Effects / Motion |
| 压缩优化 | HandBrake |

### 配音建议

- **语速**：适中偏快（技术观众接受度高）
- **语气**：热情但不夸张
- **重点词**：Token、省钱、微信、开源 等关键词加重音

### 背景音乐

- 开场：轻快的科技感音乐
- 演示：轻背景音乐，不干扰解说
- 结尾：渐强，激发行动
